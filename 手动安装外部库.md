# 手动安装外部库

如果在执行时遇到外部库自动安装失败的情况，请按照本文件中的内容手动安装。

## 1.获取Python版本

打开运行[`app.py`](app.py)使用的Python虚拟环境，输入：

~~~bash
import sys
sys.executable
~~~

将输出的内容复制下来。

## 2.手动安装

上面输出的内容这里用“python”代替。

在终端输入：

~~~bash
python -m pip install --upgrade pip
python -m pip install simpleeval flask pywebview
python -m pip install sympy numpy
~~~

## 3.测试是否成功

在使用同一Python虚拟环境的Python中输入：

~~~python
import simpleeval,flask,pywebview
~~~

如果没有报错，表示安装成功，可以运行程序了。
>如果报错，请重新尝试。如果反复报错，请[提issues](https://github.com/6871296/calculatorMax/issues/new)。

长期不使用此程序时，可以运行`librm.py`删除外部库。
