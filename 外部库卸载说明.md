# 外部库卸载说明

本程序依赖外部库`easygui`，如需卸载，请按以下步骤操作。

## 自动卸载

1. 在同一Python虚拟环境运行`librm.py`即可自动卸载。
2. 如果输出“Removation failed!”，请按下面“手动卸载”的步骤操作。

## 手动卸载

如果自动卸载失败，请按下列步骤手动卸载：

1. 打开你运行`main.py`时的Python虚拟环境，输入：

  ```python
  import sys
  sys.executable
  ```
  
将输出的内容复制下来。

2. 在终端中输入：

  ```bash
  (粘贴） -m pip uninstall easygui
  ```

3. 如果还是失败（任何步骤），请按照下面的步骤更新环境变量。
